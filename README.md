This is another project that I created (vibe coded) through ZotGPT (UC Irvine's free AI tool for students, faculty, and staff). Even though I created this with ZotGPT, it is in no way, shape, or form representative of UC Irvine's views. 

Networking is a term that gets thrown around quite a lot on LinkedIn and in the job search and in my daily work as a Career Counselor (and recruiting). Networking is very strong; in my opinion, it is the strongest tool in the job search and perhaps the least well utilized. I wanted to create this tool because I think people mistake or confuse "networking" as a behemoth task when it really boils down to making connections (genuine when possible). There are a lot of nuances within networking which are impossible to entirely capture, but I think I did a decent job covering a lot of them!

For this project, I needed to write a 1,400 word overview with most of the networking flowchart completed. I used ZotGPT to then create a design document with the full flowchart and different modifiers for routes/paths. The design document, and what was sent to ZotGPT to generate the initial HTML, was 3,600 words to generate 1,400 lines of code. Afterwards, I edited the HTML mostly for phrasing and changing many of the ENDs.

Thanks for reading and let me know if there are any issues,
